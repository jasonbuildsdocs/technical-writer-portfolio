## How do I start a new project?
Simply visit **https://bolt.new** and enter a prompt describing what you want to build.

## Do I need to install anything?
No. Bolt runs entirely in your browser using WebContainers.

## Can I edit the generated code?
Yes. All generated files are fully editable.

## How do I update a feature?
You can ask Bolt using a natural language prompt, or edit the code manually.

## How do I share my project?
Use the export or share options in the upper-right corner of the interface.

## What browsers are supported?
Latest versions of Chrome, Edge, and Firefox (recommended).
